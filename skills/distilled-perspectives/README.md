# 🧠 蒸馏视角技能 (Distilled Perspectives)

> 使用女娲技能（Nuwa Skill）蒸馏的思维框架库

## 概述

这些技能通过 AI 多源研究，提取了各领域顶尖人物的**思维模型、决策启发式、表达DNA**，可以用来"像他们一样思考"。

## 使用方式

每个技能目录结构：
```
[person]-perspective/
├── SKILL.md                    # 核心技能文件
└── references/
    └── research/
        ├── 01-writings.md      # 著作与系统思考
        ├── 02-conversations.md # 访谈与对话
        ├── 03-expression-dna.md # 表达DNA
        ├── 04-external-views.md # 外部评价
        ├── 05-decisions.md     # 决策记录
        └── 06-timeline.md      # 时间线
```

## 蒸馏流程（女娲 Nuwa）

1. **多源研究** — 6 个维度并行搜索（著作、访谈、表达、外部评价、决策、时间线）
2. **框架提取** — 提取核心思维模型、决策启发式、表达DNA
3. **技能生成** — 生成可运行的 SKILL.md
4. **质量验证** — 已知测试 + 边界测试 + 声音测试

## 注意事项

- 每个技能都有**诚实边界**，明确标注了局限性
- 内部矛盾被保留，未被美化
- 思维框架 ≠ 本人，不能替代真实判断
- 建议定期更新（特别是近12个月的新发展）
