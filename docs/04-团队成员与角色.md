## 角色职责

### CEO (马斯克)

- **战略决策**: 产品方向、技术路线图审批
- **资源决策**: 人员配置、预算分配
- **里程碑审批**: Sprint Review 参与、重大版本发布审批

### 项目经理 (刘强东)

- **项目计划核对**: 审核 Sprint 计划、任务分配合理性
- **进度跟踪**: 每日站会主持、阻塞问题升级
- **风险管理**: 识别进度风险、协调资源
- **质量把关**: 验收标准确认、Sprint 回顾组织

### 技术负责人 (moqun188)

- **Sprint 规划**: 制定迭代计划，分配任务，维护 Backlog
- **架构决策**: 技术选型、模块设计、代码评审
- **发布管理**: 版本发布、CHANGELOG 维护、Tag 管理
- **质量把控**: PR Review、验收测试、Bug 定级
- **协调沟通**: 每日站会、Sprint 回顾、跨模块协调

### 开发工程师 (sxbwork)

- **功能开发**: 按 Sprint 任务完成编码
- **代码质量**: 编写单元测试、遵循代码规范
- **Bug 修复**: 修复分配的 Bug 任务
- **文档更新**: 更新涉及模块的文档
- **代码评审**: 参与 PR Review，互相 review
